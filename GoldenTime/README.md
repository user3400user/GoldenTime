- **01_Strategie/** — STATE.md (Master), Lead-Qualitaets-Spec, Architektur-Entscheid, Geschaeftsmodell-Canvas, CC-Build-Briefing, Session-/Memory-Management, Research-Auftraege, Sprint-Plan, Onepager

## Arbeitsweise

Wir verkaufen den Kaufmoment: Gewerbebetriebe, die gerade eine PV-Anlage ohne Speicher angemeldet haben — woechentlich, qualifiziert, exklusiv pro Region an einen einzigen Speicher-Anbieter.
